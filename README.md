# AI-Powered Retail Chatbot

An AI-powered chatbot designed to automate retail customer service operations such as order tracking, refunds, cancellations, and product enquiries.

## Project Overview

This project was developed as part of my MSc Data Science dissertation at the University of Salford.
The system integrates conversational AI with a CRM database and analytics dashboard to improve customer support efficiency.

## Key Features

* GPT-powered conversational chatbot
* Order tracking and refund handling
* Product enquiry automation
* CRM request logging
* Analytics dashboard for customer interactions

## Tech Stack

Backend:

* Python
* FastAPI
* Node.js

Frontend:

* React.js

Database:

* Microsoft SQL Server

AI Integration:

* OpenAI GPT API
* Natural Language Processing (NLP)

Data Analytics:

* Power BI

## System Architecture

Customer Interface → Backend API → GPT Model → SQL Database → Analytics Dashboard
